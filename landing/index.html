Quick answers

What do I need to run it?

macOS. To build from source: Go 1.22+, Node 18+, and the Wails CLI. AI features need a local Ollama install (defaults to localhost:11434).

Is it another Electron app?

No; JumpStart is built with Wails (Go + native WebKit), with a native titlebar, translucent sidebar, and macOS appearance preferences.

What happens to processes when I quit?

Closing JumpStart cleanly stops all managed processes. Stop sends SIGTERM to the process group, then SIGKILL after five seconds.

Where is my data?

Locally, in ~/.jumpstart/config.json. Programmatic imports read from ~/.jumpstart/import.json.
