What do I need to run JumpStart?

Just macOS. To build from source you need Go 1.22+, Node 18+, and the Wails CLI. AI features use a local Ollama install, defaulting to localhost:11434.

Is it another Electron app?

No. JumpStart is built with Wails on Go and native WebKit, with a native titlebar, translucent sidebar, and macOS appearance preferences.

What happens to my processes when I quit?

Closing JumpStart cleanly stops all managed processes. Stop sends SIGTERM to the process group, then SIGKILL after five seconds.

Where is my data stored?

Locally, in ~/.jumpstart/config.json. Programmatic imports are read from ~/.jumpstart/import.json. Nothing syncs to a server.
